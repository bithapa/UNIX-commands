# push-pull-merge
push, pull, & branch

# To see the Status:

`git status`

# To add a file FILE.file:

`git add FILE.file`

# To commit:

`git commit -m "commit message - xyz changes made"`

# To crate a new branch `feature-x`

`git branch feature-x`

# To Create new branch `feature-x` & switch to it:

`git checkout -b feature-x`

# To switch back to `master`:

`git checkout master`

# To switch back to `feature-x`:

`git checkout feature-x`

# To see all of your branches:

`git branch` [* will appear next to your current branch]

# To Delete the branch `feature x`:

`git branch -d feature-x`

# To push the changes to the branch 'feature-x':

`git push origin feature-x`

# To push the changes to `master`:

`git push origin master`

# To view the history of current branch:

`git log`

# To remove a file FILE.file:

`git rm FILE.file`
